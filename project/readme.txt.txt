Open cmd and go to the directory where the project folder is present(project folder).

Create virtual environment using the command : "python -m venv myenv". This will create the virtual environment in the same directory. 

After creating, activate the virtual environment by using the command: "activate". make sure your current directory is scripts folder present inside the virtual environment,"myenv".

once activated, install all the requirements for the project using the command: "pip install -r requirements.txt"

Once all the requirements are installed, run the app with the command: python app.py

you will see message like this : Running on http://127.0.0.1:5000
copy "http://127.0.0.1:5000" and open in your browser and access the app:)
